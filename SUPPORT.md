# Support Policy

Thank you for using `SecureGoDeps`. 

This is a self-served utility project. Beyond the detailed documentation provided in this repository, there is no official help, support line, or dedicated maintenance team.

## Documentation Resources

For setup, usage, custom integrations, or troubleshooting, please refer directly to the existing README files:

1.  **Main Setup & CLI Guide**: Check the root [**`README.md`**](README.md) for utility features, prerequisite installations, CLI options (including `--dry-run`), and example terminal outputs.
2.  **CI/CD & Testing Guide**: Check the [**`examples/README.md`**](examples/README.md) for step-by-step instructions on setting up GitHub Actions workflows and running local verification tests in the `examples/testing/` directory.

If you encounter issues or have questions, please review these files carefully, as they contain all supported workflows and configuration details.
